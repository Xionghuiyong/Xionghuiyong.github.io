发布Hexo博客

Github+Nodejs+Hexo搭建博客

1.安装Git for Windows: git

2.安装nodejs: node

3.安装Hexo：npm install hexo -g

–升级Hexo： npm update hexo -g

4.初始化Hexo：Hexo init

5.使用记事本修改Hexo目录中的_config.yml文件最后位置

deploy:

–type: git

–repo: 这里填入你之前在GitHub上创建仓库的完整路径

–branch: master

5.新建博客：hexo new “新博客”

-简写：hexo n “新博客”

6.生成博客：hexo generate

-简写：hexo g

7.部署到Github:hexo depoly
-简写：hexo d
